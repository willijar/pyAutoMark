.. _Subcommand find-duplicates:

find-duplicates
===============

Synopsis
--------

**pyam find-duplicates** [-h] [--cohorts ...]
                
Description
-----------

:program:`pyam find-duplicates` finds duplicate files from students across cohorts

Options
-------

.. program:: pyam find-duplicates

.. option:: -h, --help

   Show a help message and exit

.. option::  --cohorts ...
    
   List of cohorts to scan across - defaults to current only
