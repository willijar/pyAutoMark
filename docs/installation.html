Overview¶

pyAutoMark is written in `Python`__ and supports Python 3.8+. It is built upon the `pytest`__ unit testing framework and `openpyxl`__ which must be installed first.

Installation from source¶

Currently you must install pyAutoMark directly from a clone of the `Git repository`__. This can be done either by cloning the repo and installing from the local clone, on simply installing directly via git.

$ git clone https://github.com/willijar/pyAutoMark
$ cd pyAutoMark
$ pip install .
